# Vant Public Lessons

## Verbosity Modes
- lib/verbosity.js controls output (full/terse/quiet)
- models/v0.8.1/verbosity.ini persists setting
- Set with: verbose.set('terse') or config file

## Key Files
- bin/vant.js - CLI
- lib/verbosity.js - Output control
- dist/index.html - Landing page

## Notes
- Run from project root: node bin/vant.js [command]
- Use full mode for debugging, terse for production
