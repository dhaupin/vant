## About

Vant solves: AI agents lose all context when sessions end.

**Why Git?**
- Versioning built in
- Branches for experiments
- Pull requests for reviews
- Distributed sync

Use Vant for long-running agentic workflows, multi-agent systems, and persistent AI memory.
