# VANT CLI

A lightweight CLI tool for AI agent session management with GitHub-backed state.

## Installation

```bash
npm install -g vant
```

## Usage

```bash
vant start        # Health check → sync → run
vant sync        # Pull latest from GitHub  
vant health      # System diagnostics
vant status      # Current status
```

This is the CLI skeleton. Full system at: https://github.com/dhaupin/VANT
